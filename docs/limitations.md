# Limitations

이 Skill Kit이 할 수 없는 것, 또는 사용자가 반드시 책임져야 할 영역.

- **완전 자동화 도구가 아니다.** plan 승인, My Note 작성, Review 결과 반영은 사람의 판단이 들어가야 동작한다.
- **AI 결과는 반드시 검토가 필요하다.** Summary와 Draft가 자료를 정확히 반영했는지 사용자가 확인해야 한다.
- **해석은 사용자가 직접 작성해야 한다.** My Note를 AI에게 떠넘기는 순간 이 Kit의 목적이 무너진다.
- **README Draft는 완성본이 아니라 초안이다.** GitHub README에 그대로 붙이지 않는다. 사용자가 다듬은 뒤 사용한다.
- **공식 문서나 최신 정보가 필요한 경우 별도 확인이 필요하다.** AI가 만든 Summary는 자료 시점에 묶이며, 시간이 지나면 사실관계가 달라질 수 있다.
- **Codex의 실제 인식 경로는 환경마다 다를 수 있다.** `.agents/skills/` 위치가 모든 Codex 실행 환경에서 인식되는지는 사용 전 확인이 필요하다.
- **본 Kit은 한국어 기준으로 설계되었다.** 다른 언어로 사용하려면 템플릿 번역과 가드레일 검토가 필요하다.
- **본 Kit은 문서·템플릿·Skill 정의에 한정된다.** 외부 자료 크롤링, API 연동, CI 통합은 포함되지 않는다.
